# adstir Integration Guide for Unity

adstirのインテグレーションガイドです。Unityプラグインは管理画面から入手することができます。

## 前提条件
* Unity2017 以上

### Android
* Android 4.4 以上
* minSdkVersion 14以上
* compileSdkVersion 28以上
* AndroidX 必須

### iOS

* iOS 9 以上
* Xcode 10 以上
* CocoaPods 必須

## Unityプラグインのインポート

[初期設定](init.md)からUnityプラグインをプロジェクトへインポートしてください。
